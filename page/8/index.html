X.Org Server 1.11正式发布

Posted on August 27, 2011

X.Org Server 1.11正式版于昨日晚间发布了。因为发布管理员Keith Packard的母亲不幸去世(节哀顺变)，新版发布比原计划推迟了一周。这是一次比较重大的更新，汇聚了过去六个月的增强，并修复了大量bug，但却没有带来什么新特性。支持触摸操作的X Input 2.1被推迟到了下一个X.Org Server版本或者更晚，也没有任何RandR扩展。

发布说明也非常简单：“最终的1.11版在此。一如既往地感谢各位对代码、测试、归档和检查的贡献。”

X.Org Server 1.11不会集成在Ubuntu 11.10中，但会在Fedora 16和其它发行版中出现。

X.Org Server 1.12按惯例会在半年后发布，目标之一是支持平滑滚动。下个月的XDC Chicago 2011大会上，开发人员们可能会就此新版本展开讨论。

X.Org 7.7的发布时间倒是个问题。上个版本7.6是去年年底发布的，新版仍然只是模糊地定在2011年下半年，很有可能要等到2012年初。

Linux 3.1内核将支持无线局域网唤醒

Posted on August 17, 2011

Linux操作系统早已支持局域网唤醒(WOL)，而下一版本的Linux 3.1 Kernel系统内核将会引入无线局域网唤醒，简称WWOL。和有线版本类似，无线局域网唤醒可以让系统在进入挂起状态(ACPI S3)或者另一种低功耗状态的同时继续保持无线连接处于开启状态，而一旦接到用户的特定操作，或者从AP断开连接，就会将系统唤醒，并恢复到原始状态。

该技术也可以用于在无线网卡找到合适的Wi-Fi连接时自动唤醒系统。

Linux 3.1对无线局域网唤醒的支持来自于五月份就发布的一个补丁，并最终在merge window阶段成功进入了主干树(mainline tree)。Intel也在他们的iwlagn驱动中加入了对该技术的支持。

Linux 3.1内核正式版将于下个月发布，新功能除了无线局域网唤醒之外还有Nouveau驱动在GeForce 400/500系列显卡上无需手动载入微代码、Sandy Bridge性能优化、大量DRM改进、支持开源处理器架构OpenSIRC、Wii Remote HID驱动、Intel Atom平台Poulsbo驱动改进和Cedar Trail初步支持。

2011年最佳的 Linux 发行版

Posted on August 14, 2011

新闻来源:开源中国
这类的文章很无聊，但也很受欢迎，不过本文有点不同，主要为你推荐不同的应用场景下的最佳的 Linux 发行版本，没有之一。
1. 最佳的 Linux 桌面发行版 —— Fedora Linux 在过去几年一直都是 Ubuntu 和其衍生版本占据 Linux 桌面的头把交椅，但自从 Ubuntu 11.04 发布之后饱受诟病以及 Fedora 15 的发布，使得 Fedora 在今年拔得头筹。

2. 最适合笔记本的 Linux 发行版 —— Ubuntu
Ubuntu 和 Fedora 本来不相上下，但因为有了 Ubuntu One 使得 Ubuntu 在移动性上更胜一筹。
Ubuntu One是由 Ubuntu 背后的公司 Canonical 所推出的一项网络服务。该服务能够存储你的文件，并允许你在多台电脑上同步，还可以与好友分享这些文件。你只需指定欲分享用户的Email地址，该用户就 会受到邀请邮件，接受验证后改文件就会自动出现在其“Shared With Me”文件夹中。

3. 最佳的企业级 Linux 桌面发行 —— SUSE Enterprise Linux
原本 SUSE 和红帽的企业Linux在企业级桌面上势均力敌，二者在功能、性能上几乎相同，但 SLED 受益于其 openSUSE Build Service ，通过该免费开放的服务使得开发者可轻松构建基于 SUSE 和openSUSE 平台的应用。
这在企业开发市场上绝对受欢迎。

4. 最佳企业服务器版本 —— RHEL
我想大家对红帽企业 Linux 摘此桂冠应该没有什么异议，加之最近刚发布的 6.0 乃至 6.1 版本，更为其增加了不少新粉丝，而基于 RHEL 的 CentOS 也同属此序列。
5. 最佳的 Linux LiveCD —— KNOPPIX
国内应该很多人对 KNOPPIX 不太熟悉，KNOPPIX是 一套光盘启动的GNU/Linux系统(LiveCD)， 功能包括：自动硬件监测、支持常见的显卡、声卡、SCSI和USB设备，以及其它外设。KNOPPIX可用于Linux演示、光盘教学、系统急救，经过适 当改造，还可以用于商业软件的产品演示。KNOPPIX采用了特殊的解压缩技术，不需要硬盘安装，一张CD，就容纳了2GB的可执行程序，供用户自由使 用。

6. 最佳的 Linux 安全发行版 —— BackTrack Linux
BackTrack是基于Ubuntu的自启动运行光盘，它包含了一套安全及计算机取证工具。它其实是依靠融合Auditor Security Linux和WHAX（先前的Whoppix）而创建成的。
BackTrack是目前为止知名度最高,评价最好的关于信息安全的Linux发行版.它是基于Linux平台并集成安全工具而开发成的Linux Live发行版,旨在帮助网络安全人员对网络骇客行为进行评估.不论您是否把它做为常用系统,或是从光盘或移动硬盘启动,它都可以定制各种安全包, 包括内核配置,脚本和补丁, 以用于入侵检测.

7. 最佳多媒体 Linux 发行版 —— Ubuntu Studio
Ubuntu Studio 是一个 Ubuntu 的变种版本，主要是提供一个捆绑了专业音频、视频和图形处理软件的工作站版本。
